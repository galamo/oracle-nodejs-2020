## NVM installation
- MAC: `brew install nvm`
- Windows: `https://docs.microsoft.com/en-us/windows/nodejs/setup-on-windows`

# Run Nodejs
- create folder
- create `index.js` file
- insert some JS code inside the file
- `node index.js`


# Installations
- `npm install nodemon -g`
- `npm i express`
- `npm install body-parser`
- `npm install dotenv`
